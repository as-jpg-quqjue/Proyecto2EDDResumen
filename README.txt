Este proyecto de Java tiene la funcionalidad de simular una cola de impresión mediante un montículo binario, el cual permite modificar o imprimir
objetos en cola con mayor o menor prioridad. El programa esta diseñado para demostrar de manera visual como funciona una cola de este tipo.

Se puede añadir y eliminar usuarios, añadir documentos a estos usuarios, enviar estos documentos a la pila de cola con diferentes prioridades según el documento
y el usuario, se puede visualizar todos los documentos y usuarios. También se puede visualizar la cola de impresión como una cola simple o como un arbol y
se puede añadir o eliminar documentos de la cola de impresión.
